# AI 对话记录助手

> 自动记录你的每一次 AI 对话 · Chrome 插件

**AI 对话记录助手**（AI Conversation Recorder）是一款 Chrome 浏览器扩展，支持在 **通义千问**、**豆包** 等主流 AI 平台对话时，**自动捕获并保存**对话内容到本地浏览器。数据优先本地存储，可选云端同步备份，支持全文搜索与多格式导出。

- 官方网站：[https://chatai.fan/](https://chatai.fan/)
- 插件目录：`AIChromePlug/`（加载此文件夹安装插件）

## 核心功能

### 🎯 自动捕获对话

- 安装后**无需手动复制粘贴**，在千问、豆包页面正常对话即可
- 后台静默监听对话变化，**实时增量保存**到浏览器本地存储
- 页面右下角提供 **💾 手动保存** 浮动按钮，可随时强制保存当前对话
- 保存成功时显示绿色提示（可在设置中关闭）

### 🔍 全文搜索与对话管理

- 点击浏览器工具栏插件图标，打开 **Popup 弹窗**查看所有已保存对话
- 支持按**标题、正文、平台、标签**全文搜索
- 点击对话可查看完整消息记录
- 支持为对话**添加标签**、**删除**不需要的记录
- 列表显示平台来源（千问 / 豆包）、消息数量、保存时间
- ☁️ = 已同步到云端 · 📱 = 仅保存在本地

### ☁️ 云端同步备份（可选）

- 开启云端同步后，对话自动备份到服务端（默认 [https://chatai.fan](https://chatai.fan)）
- 换设备、重装浏览器后，登录同一账号即可恢复数据
- 支持**手动从云端拉取**，与历史导出工具上传的数据互通
- 每 5 分钟自动同步未上传的对话（可在设置中配置）

### 📤 多格式导出

| 格式 | 说明 |
|------|------|
| **Markdown** (.md) | 单条或全部对话，适合笔记与二次编辑 |
| **PDF** (.pdf) | 排版美观，适合归档与分享 |
| **JSON** | 完整数据备份，可导入恢复 |

导出时可选择是否包含时间戳、角色标识（用户 / AI）。

### 🔒 隐私与敏感词过滤

- 数据**默认只保存在本地**，不上传任何服务器
- 只有你主动开启云端同步后，数据才会传输到配置的服务端
- 支持**敏感词过滤**：含「密码」「token」等关键词的消息可自动跳过不保存
- 过滤策略可选：跳过单条消息 / 跳过整段对话 / 仍然保存

### 💎 灵活套餐

- **免费版**：无限本地存储 + 每月 10 条云端同步 + Markdown 导出
- **付费版**：更大云端配额、PDF 导出、优先客服等高级功能
- 在插件设置页 → **升级套餐** 查看详情与购买

## 安装指南

> 需要 **Chrome 88+** 或其他基于 Chromium 的浏览器（Edge、Arc 等）。

### 方式一：Chrome 应用商店（推荐）

1. 打开 [Chrome 应用商店](https://chrome.google.com/webstore)
2. 搜索 **「AI 对话记录助手」**
3. 点击 **「添加至 Chrome」** → 确认权限
4. 安装完成后，工具栏出现插件图标

### 方式二：本地加载（开发者模式）

适用于本仓库中的 `AIChromePlug` 文件夹：

1. **解压或定位插件目录**  
   确保 `AIChromePlug/` 文件夹内包含 `manifest.json`（本仓库已就绪）。

2. **打开 Chrome 扩展程序页面**  
   地址栏输入并回车：
   ```
   chrome://extensions
   ```

3. **开启开发者模式**  
   页面右上角打开 **「开发者模式」** 开关。

4. **加载插件**  
   点击 **「加载已解压的扩展程序」**，选择 `AIChromePlug` 文件夹（不是 zip 文件本身）。

5. **固定到工具栏**  
   点击工具栏 🧩 拼图图标 → 找到 **「AI 对话记录助手」** → 点击 📌 固定。

6. **验证安装**  
   打开 [通义千问](https://www.qianwen.com/) 或 [豆包](https://www.doubao.com/)，发送一条对话 → 点击插件图标 → 确认对话出现在列表中。

> ⚠️ 本地加载的插件在 Chrome 大版本更新后可能需要重新启用；请勿删除或移动插件文件夹。

## 使用步骤

### 第一步：安装并固定插件

按上方 [安装指南](#安装指南) 完成安装，将插件图标固定到浏览器工具栏。

### 第二步：正常使用 AI 平台

1. 打开 **通义千问** 或 **豆包** 网页
2. 像往常一样与 AI 对话
3. 插件在后台**自动捕获**对话内容，无需任何额外操作
4. 若看到右下角 **「✓ 已保存 N 条消息」** 绿色提示，说明保存成功

**手动保存（可选）：** 点击页面右下角的 **💾 按钮**，可强制保存当前对话。

### 第三步：查看与管理对话

1. 点击浏览器工具栏的 **插件图标**，打开 Popup 弹窗
2. 浏览 **最近保存** 的对话列表
3. 在顶部搜索框输入关键词，**全文搜索**标题、正文、平台或标签
4. 点击某条对话，查看完整消息记录
5. 点击 **⚙️ 设置** 进入完整设置页（或右键插件图标 → 选项）

### 第四步：导出对话（可选）

**在 Popup 中：**

1. 打开对话详情或点击导出入口
2. 选择格式：**Markdown** 或 **PDF**
3. 勾选是否包含时间戳、角色标识
4. 选择 **导出当前对话** 或 **导出全部**

**在设置页中：**

1. 进入 **数据管理** 区域
2. 点击 **导出备份**（JSON 完整备份）
3. 点击 **导出 MD** 或 **导出 PDF**

### 第五步：开启云端同步（可选）

1. 打开插件 **设置页**（Popup 右上角 ⚙️，或 `chrome://extensions` → 插件详情 → 扩展程序选项）
2. 展开 **云端同步** 区域
3. 填写 **服务端地址**（默认 `https://chatai.fan`，自部署用户填自己的地址）
4. 点击 **测试连接** 确认网络正常
5. 输入 **邮箱** → **发送验证码** → 登录
6. 开启 **云端同步** 和 **自动同步**
7. 点击 **从云端拉取**，可将服务端已有数据同步到本机

> 使用 [历史导出工具](https://chatai.fan/#export-tool) 上传的数据，也须用**相同服务端地址和邮箱**登录后拉取。

## 常见问题

**Q：支持哪些 AI 平台？**  
A：目前支持通义千问、豆包。更多平台持续适配中。

**Q：数据安全吗？**  
A：默认只保存在浏览器本地。只有你主动开启云端同步后，数据才会传输到你配置的服务端。支持自部署服务端。

**Q：免费版有什么限制？**  
A：免费版可无限本地保存，每月云端同步 10 条，支持 Markdown 导出。个人轻度使用完全够用。

**Q：插件会影响网页速度吗？**  
A：不会。插件仅在 AI 平台页面注入轻量脚本，不影响正常交互。

**Q：Chrome 更新后插件失效？**  
A：本地加载的插件可能需要在 `chrome://extensions` 重新启用，或再次「加载已解压的扩展程序」。

**Q：云端同步配额不足？**  
A：在设置页查看套餐配额，或点击「升级套餐」购买更大配额。

**Q：如何联系作者？**  
A：访问 [https://chatai.fan/](https://chatai.fan/) 查看联系方式与更多文档。
